# Katoetal2023

This study can be reproduced with the files listed bellow. Note: Before running the code, public datasets must be stored according to the texts in the files and the directories must be adjusted appropriately for the environment. All the codes were run on an iMac Pro (2017) with macOS Mojave (2.3GHz Intel Xeon W, 256 GB 2666 MHz DDR4). Note: the Fortran code for HBA calculation from NuPoP (ver. 1.20.0, GPL-2) was adapted for this study.
